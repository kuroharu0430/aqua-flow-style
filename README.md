## 1. Overview（概要）

このアプリケーションは、  
**Web上で要素を自由に配置・編集できる UI エンジンベースのレイアウト編集ツール**です。

グリッド上にパーツをドラッグして配置し、  
サイズ変更、複数選択、スタイル編集、Undo/Redo など  
直感的な操作でレイアウトを構築できます。

Webブラウザ上で動作し、  
Blazor / C# による高い操作性と柔軟な UI 制御を実現しています。

## 2. Features（機能一覧）

- **ドラッグ移動（Drag）**  
  要素をグリッド上で自由に移動できる。

- **リサイズ（Resize）**  
  右下のハンドルを掴んでサイズ変更が可能。

- **新規配置（Registering）**  
  テンプレートを選択してグリッドに新規配置。

- **複数選択（Selecting）**  
  範囲選択で複数の要素を一括操作。

- **コンテキストメニュー（右クリック）**  
  編集・削除などの操作を提供。

- **スタイル編集（Style Edit）**  
  ダイアログから色・枠線・フィールド値などを編集。

- **Undo / Redo（操作履歴）**  
  すべての操作が履歴として記録され、戻す・やり直しが可能。

- **オーバーラップモード（Push / Free）**  
  要素同士の衝突時の挙動を切り替え可能。

- **Trail Effect（移動軌跡）**  
  移動中のグリッドにエフェクトを表示。

## 3. Technical Background（技術背景）

このアプリケーションは、Blazor 上で高度な UI 操作を実現するために  
いくつかの技術的課題を解決しています。

  ### 1. Blazor のレンダリング制約を超える Drag/Resize
  Blazor は DOM 操作が制限されているため、  
  通常のドラッグ処理は実装が困難です。
  
  本アプリでは、  
  - JS との連携  
  - 60fps の DragLoop  
  - State を中心とした再描画  
  により、滑らかなドラッグ操作を実現しています。
  
  ### 2. Surface / Controller / State の三位一体構造
  UI の見た目・操作の意味・状態の真実を分離することで、  
  複雑な UI 操作をシンプルな構造で扱えるようにしています。
  
  ### 3. Undo / Redo のための Snapshot 設計
  操作履歴を「差分」として記録することで、  
  正確な Undo / Redo を実現しています。
  
  ### 4. 座標系の完全抽象化
  Page / Surface / Scroll / Relative / Absolute の座標を統合し、  
  Controller が座標計算を意識せず操作できるようにしています。
  
  ### 5. Session による操作文脈の管理
  Drag や Selection の開始位置・対象・影響範囲を Session として保持し、  
  操作の文脈を安全に管理しています。


- **グリッドレイアウト管理**  
  Column / Row / Cell サイズを柔軟に設定可能。
